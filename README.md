# Data Analysis Basic — Titanic Dataset

This project is a simple and clean end-to-end data analysis workflow using the Titanic dataset.

## Goals
- Load & explore dataset  
- Clean data (missing values, types, features)  
- Perform EDA (visualizations, distributions, correlations)  
- Build a simple ML baseline model (e.g. Decision Tree)  
- Document the entire workflow in a clear Jupyter notebook  

## Project Structure
/data               → raw CSV files  
/notebooks          → Jupyter notebooks  
/scripts            → Python helper scripts  
/assets             → images, plots  
requirements.txt    → dependencies  
README.md           → project documentation  

## Technologies
- Python
- pandas
- numpy
- matplotlib / seaborn
- scikit-learn
